# Professional Password Generator

A highly configurable, professional-grade GUI application built with Tkinter for generating, managing, and storing secure passwords.

## Features

- **Personalized Generation**:
  - Enter a name and choose to include it in the password
  - Name is inserted at a random position for personalization

- **Password Strength Meter**:
  - Real-time strength calculation
  - Visual progress bar with color coding (Weak/Medium/Strong/Very Strong)

- **Professional Interface**:
  - Modern ttk styling with Segoe UI fonts
  - Organized layout with frames and grids
  - Menu bar (File, Tools, Help)
  - Status bar with feedback
  - Responsive design

- **Management Features**:
  - Generate and copy passwords instantly
  - Save to history with double-click to copy
  - Save/load password lists (JSON format)
  - Export history as text file
  - Clear history option

## Requirements

- Python 3.x
- Tkinter (built-in)

## Installation

Run the script:
```
Modern_Password_Generator_Tool.py
```

## Usage

1. Configure password options in the Configuration section.
2. Set requirements if needed.
3. Click "Generate Secure Password".
4. View strength indicator and copy/save as needed.
5. Use history for previously generated passwords.
6. Save/load lists via File menu, or export via Tools menu.
